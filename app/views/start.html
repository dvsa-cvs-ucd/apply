Before you start

Some applications still require you to complete and upload an application form. Decide which form you need by looking at the forms below:

  • Application forms for a coach or bus
  • Application forms for a lorry or trailer
  • Application forms for a motorcycle or three-wheeled vehicle
  • Application forms for a passenger car

If you have a 'Manage your vehicle testing' account, sign in to that service to:

  • make applications
  • track applications
  • make payments

If you want to make more than one application, we recommend you create an account. This is so you can:

  • save time in making applications for the same or similar vehicles
  • track applications

You can also apply via email or by post.
